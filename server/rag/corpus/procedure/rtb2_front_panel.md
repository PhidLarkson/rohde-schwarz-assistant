Procedure: R&S RTB2 Front Panel Overview

The R&S RTB 2 Series (RTB22 2-channel, RTB24 4-channel) front panel layout:

Input connectors (bottom of front panel):
- CH1 to CH4: BNC analog input channels (RTB24 has 4, RTB22 has 2)
- Ext. Trigger In: external trigger input (BNC)
- Probe Comp: probe compensation output (1 kHz square wave for probe calibration)
- Aux Out: auxiliary output
- Logic Channels: 16 digital channels for MSO option (D0-D15)
- Pattern Generator: digital pattern output pins (P0-P3, optional R&S RTB-B6)
- USB port: for saving screenshots and waveform data to USB flash drive

Front panel keys:
- Run Stop: starts/stops continuous acquisition
- Single: triggers a single acquisition
- Autoset: automatic waveform detection and display optimization
- Default: resets to factory default settings
- Horizontal controls: Scale knob (time/div), Position knob (trigger offset)
- Vertical controls: Scale knob (V/div), Position knob (offset) per channel
- Trigger controls: Level knob, Source button, Slope button, Mode (Auto/Normal)
- Menu keys: Measure, Cursor, Math, Search, Analysis
- Action keys: Save, Quick Save (OneTouch)

Rear panel:
- LAN port (RJ45): for remote control via Ethernet/TCPIP
- USB Device port: for remote control from a PC via USB TMC
- USB Host port: for USB peripherals
- GPIB port (optional): IEEE 488 remote control

Display: 10.1 inch capacitive touchscreen (1280x800)
The display shows waveforms, measurement results, status bar with channel info, timebase, trigger status, and sample rate.
